# Employee Turnover Prediction

Machine Learning project to predict employee attrition using Random Forest, Gradient Boosting, and Logistic Regression.

## Dataset
The dataset contains employee information such as satisfaction level, evaluation score, number of projects, working hours, and salary.

## Techniques Used
- Exploratory Data Analysis (EDA)
- Correlation Heatmap
- K-Means Clustering
- SMOTE for class imbalance
- Logistic Regression
- Random Forest
- Gradient Boosting
- ROC Curve & AUC

## Best Model
Random Forest achieved the best performance with ~99% accuracy.

## Business Insight
Employees were classified into risk zones to help HR identify employees likely to leave and take preventive actions.
